This file has two AJAX requests that read from .json files. One is for the employee status list, and the other is for the room availability list. In a real usage of this sort of thing, the .jason file would be generated dynamically on the server, but this is for demonstration purposes.

JSON is very tricky. Both AJAX requests and JSON files come through OK on my server, but the second script will not run locally.

Also, when doing this, it may be a good idea to use absolue file paths in the javascript, because these are not always handled the same way.

If in the script, a server status is checked in the conditional, like in this one, the if statement checks for a server status of 200 (request successful) in order to output the data on the html page, this will not run on a local machine, as only a server can return a server status. So if you want to run something like this locally, you will need to remove the server status check from the if statement (or anywhere else, probably) for this to work.
